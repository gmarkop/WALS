About WALS Local Explorer

About This Application

WALS Local Explorer is a web-based application that allows you to explore the World Atlas of Language Structures (WALS) dataset on your local computer. It provides an intuitive interface for browsing languages, exploring typological features, and analyzing linguistic data.

Features

  • Language Browser: Browse and search through thousands of languages
  • Feature Explorer: Explore typological features across linguistic domains
  • Interactive Map: Visualize the geographic distribution of languages
  • Statistics: View data visualizations and statistics
  • Advanced Search: Find languages and features quickly
  • Offline Access: Works entirely on your local machine

About WALS

The World Atlas of Language Structures (WALS) is a comprehensive database of structural (phonological, grammatical, lexical) properties of languages gathered from descriptive materials. It was first published as a book with CD-ROM in 2005 by Oxford University Press.

Coverage

  • 2,679 languages from all major language families
  • 192 features covering phonology, morphology, syntax, and lexicon
  • Global coverage with languages from all continents
  • 144 authors contributing expert knowledge

Citation

If you use WALS data in your research, please cite:

Dryer, Matthew S. & Haspelmath, Martin (eds.) 2013. The World Atlas of Language Structures Online. Leipzig: Max Planck Institute for Evolutionary Anthropology. (Available online at https://wals.info)

Technology

This application is built with:

  • Flask: Python web framework
  • CLDF: Cross-Linguistic Data Format for standardized data access
  • Leaflet: Interactive maps
  • Chart.js: Data visualizations
